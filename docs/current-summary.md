# Current Project Summary

## Project Overview

**title-tk** is a task management app for neurodivergent users inspired by Jaclyn Paul's "Order from Chaos" book. The core goal is eliminating decision paralysis through structured task management.

## Key Concepts Established

### Inbox Processing

- Primary workflow: capture ideas quickly without categorization → process later when mentally ready
- Sticky note analogy: raw ideas that might become work, not formal tasks yet
- Two-minute rule: do immediately, convert to task, convert to project, or delete
- Processing sessions are separate from task execution

### System Architecture

- **Inbox items**: uncategorized ideas/thoughts
- **Tasks**: single-step actions with context, status, priority
- **Projects**: multi-step work containing multiple tasks
- **Contexts**: when/where tasks can be done (House, Computer, Errands, etc.)

### Task Properties

- Context (required)
- Status: Next Action, Waiting, Delegated
- Optional: due date, priority, blocking dependencies
- Projects can contain multiple tasks
